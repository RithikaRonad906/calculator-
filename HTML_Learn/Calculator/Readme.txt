# SimpleSUM Calculator

Welcome to **SimpleSUM**, a basic calculator web app that makes math simple and easy. Perform addition, subtraction, multiplication, and division with a straightforward and user-friendly interface.

## Features

- Basic arithmetic operations: addition, subtraction, multiplication, and division.
- Clear button (C) to reset the input field.
- Equal button (=) to calculate and display the result.
- Responsive design for various screen sizes.

## Usage

1. Enter the numbers and choose an arithmetic operation by clicking the respective buttons.
2. Click the "=" button to calculate the result.
3. To clear the input field, click the "C" button.

## Getting Started

To run the SimpleSUM calculator on your local machine, follow these steps:

1. Clone this repository: `git clone https://github.com/your-username/simple-sum-calculator.git`
2. Open the repository in a web browser.
3. Start calculating!
